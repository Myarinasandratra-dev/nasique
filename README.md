🎧 Projet Nasique
Nasique est une application web musicale inspirée de Spotify et Monstercat, développée avec React.js et Tailwind CSS. Elle permet de découvrir, écouter et explorer des albums et des morceaux dans une interface moderne et responsive.

🚀 Objectifs du projet :
Reproduire une expérience utilisateur similaire à Spotify / Monstercat

Apprendre à structurer une application React avec plusieurs pages (React Router)

Utiliser Tailwind CSS pour construire une interface propre, rapide et responsive

Organiser les composants réutilisables (Navbar, Footer, TrackList, AlbumHero, etc.)

Préparer une base pour ajouter une future base de données (ex: Supabase ou Firebase)

🧱 Stack technique :
React.js – pour construire l’interface dynamique

React Router – pour gérer la navigation entre les pages

Tailwind CSS – pour le design rapide et moderne

Vite – pour le bundling et le démarrage ultra rapide

📁 Fonctionnalités prévues :
🎵 Page d’accueil musicale avec visuel d’album

🗂️ Liste de morceaux avec bouton de lecture

🔗 Liens de streaming vers YouTube, Spotify, Apple Music

🧭 Navigation entre les pages : Accueil, Contact, Profil

🧱 Structure modulaire et évolutive (prête pour ajout backend)

🛠️ À venir :
🎧 Lecteur audio intégré

🔍 Moteur de recherche musical

👤 Connexion utilisateur et favoris

📡 Intégration à une base de données (ex: Supabase, Firebase)
